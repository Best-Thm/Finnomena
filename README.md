# Prerequisites
- flutter 2.5.3 (https://flutter.dev/docs/get-started/install)
- Android studio
# Run Code
- open vscode at project folder
- select emulator (in my case, I use `Pixel 2 API 30 (android-x86 emulator)`)
- run terminal with command 
    ```flutter run```
# Unit Test
- open vscode at project folder
- run terminal with command 
    ```flutter test```
